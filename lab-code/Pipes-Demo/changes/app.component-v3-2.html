Demonstrating Currency Pipe

Formatting currency based on locale

Showing amount using default locale of US : {{firstNum | currency : 'EUR': 'symbol' : '3.1-3' }}

Showing amount using France as locale : {{firstNum | currency : 'EUR': 'symbol' : '3.1-3' : 'fr' }}

Showing amount using Hindi as locale : {{firstNum | currency : 'EUR': 'symbol' : '3.1-3' : 'hi' }}

Showing amount using Arabic as locale : {{firstNum | currency : 'EUR': 'symbol' : '3.1-3' : 'ar' }}

Showing amount using Arabic as locale : {{firstNum | currency : 'SAR': 'symbol' : '3.1-3' : 'ar' }}
